About Universiti Teknologi Mara

Top Glove Corporation Bhd is the world's largest manufacturer of gloves, supplying protective and medical gloves globally.

During my time at Top Glove, I was involved in multiple industrial engineering projects focusing on production optimization, safety improvements, and sustainable manufacturing practices.

IoT-Enhanced Greenhouse Control System (Final Year Project)

Designed and developed a smart greenhouse system using ESP32 and various environmental sensors (DHT11, soil moisture, LDR) to monitor and control temperature, humidity, and soil conditions in real time. The system featured solar power integration, automated irrigation, fan control, and LCD-based feedback, making it energy-efficient and suitable for household farming applications.

SLAM-Based Autonomous Wheelchair Robot

Programmed a Raspberry Pi and Arduino-based robotic wheelchair that navigated autonomously using SLAM (Simultaneous Localization and Mapping) with LiDAR. The system mapped environments, avoided obstacles, and demonstrated sensor fusion and indoor navigation technologies.

Real-Time Monitoring & Control Dashboard using ESP32

Built a web-based dashboard using Python, HTML, and CSS to display sensor data such as temperature and humidity in real time. The system allowed remote control of LEDs and demonstrated fundamental IoT integration between ESP32 microcontrollers and cloud-based visualization tools.

Smart Surveillance System using Deep Learning

Developed an AI-powered surveillance system using Python, TensorFlow, and OpenCV for facial recognition and anomaly detection. The project processed live video streams to identify known individuals and flag unusual behavior, showcasing the application of machine learning in security systems.

Phishing Sentinel – Cybersecurity & AI Integration

Created an AI-based phishing detection tool in Python that analyzed URLs and textual patterns to detect potential phishing attempts. The system automated threat identification and served as a lightweight, real-time cybersecurity solution, highlighting your interest in digital systems and safety.

Basic Industrial Automation System

Developed a comprehensive virtual automation system using Factory I/O and programmed it in ladder logic using WinSPS-S7. The system included raw material sorting, lid and base machining centers, product assembly using pick-and-place, defect detection and removal, and final product sorting into an automated warehouse. Key components included item counters, wheel sorters, defect filters, lifting mechanisms, and synchronized conveyors, simulating a full end-to-end smart production line.

Mechanical Design & 3D Modelling

Created 3D models and mechanical assemblies using CATIA V5 and FreeCAD, including gear systems, linkages, and enclosures for embedded devices. The project demonstrated mechanical design proficiency and compatibility with automation system development.
